LSL Architecture Map

Living Synapse Language Model là một kiến trúc ngôn ngữ CPU-only dựa trên biểu diễn thưa, học online cục bộ, predictive coding, cortical columns, long-context memory và reasoning workspace. Mục tiêu không phải mô phỏng Transformer, mà là tạo một đường suy luận không attention matrix, không backprop và có chi phí theo token gần hằng số.

Luồng Kiến Trúc Hợp Nhất

LSL nên được xem như một model duy nhất có shared tokenizer, shared sparse state bus, nhiều mạch chuyên biệt và một unified candidate scorer. Các module không thay nhau làm model riêng; chúng cùng quan sát một token stream và cùng cập nhật trạng thái cục bộ.

Các Mạch Chính

Mỗi mạch nhận cùng token/SDR stream, trả về tín hiệu cục bộ, candidate hoặc memory state.

SDR

Sparse Distributed Representation

Mã hóa token/khái niệm thành active indices thưa. Đây là bus chung cho compute, memory và semantics.

PC

Predictive Coding

So sánh trạng thái dự đoán và trạng thái thật để tạo local error, suppression và update cục bộ.

Columns

Cortical Column Memory

Học chuỗi token, xử lý surprise bằng burst/silent dynamics, giữ độ trễ gần hằng số theo context.

Memory

Long Context + World Memory

Truy hồi fact, evidence và transition qua bounded sparse lookup, không attention matrix và không full scan.

Workspace

Reasoning Workspace

Lưu biến cục bộ, trace toán/chương trình, multi-hop relation và entity-event graph.

Bio

Neuromodulation + Dendrites

Gating plasticity, novelty, fatigue và branch-local nonlinear compute để điều chỉnh học online.

Vòng Đời Một Token

Encode. Text được tách bằng tokenizer chung, token id được mã hóa thành SDR active indices.

Predict. Columns, long-context memory, predictive stack và workspace đều trả candidate hoặc local score.

Score. Unified scorer gộp xác suất cục bộ, semantic continuity, reasoning confidence, fatigue và anti-loop penalty.

Emit. Decoder chọn token tiếp theo hoặc câu trả lời, rồi đưa token đó trở lại state stream.

Learn. Hebbian update, fatigue, hippocampal fast memory và consolidation chạy online tại active synapses.

def observe_token(token_id):
    active = sdr.encode(token_id)
    pc_error = predictive.observe(active)
    columns.forward(token_id, learn=True)
    long_context.observe_transition(prev_token, token_id)
    neuromod.gate(pc_error, novelty=active)
    synapse.forward_active(active)
    synapse.hebbian_update_active(modulator=neuromod.value)

Ràng Buộc Strict

Không Được Dùng

No backprop, no autograd, no Adam/SGD/RMSprop, no GPU requirement, no PyTorch/TensorFlow/JAX runtime, no Q/K/V attention, no all-pairs token interaction, no global hidden error.

Phải Duy Trì

Online learning, local updates, per-token state transition, active-index sparse compute, bounded candidate lookup, no full-history scan và strict structural scanner.
